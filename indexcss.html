Explicación de las cosas mas importantes de CSS


CLASES:

Las Clases son una de las cosas mas importantes del css, podemos definir una clase con class="MiClase", y en el css tenemos que declarar la clase:

.MiClase{
color: red;

}

Esto es un h2 con clase


TIPOS DE CLASES:

Los tipos de clases mas usados en CSS son:

Padding: Sirve para declarar el tamaño de separación que quieres que tenga la caja (DIV) de el texto de interior.

padding: 1cm 1cm 1cm 1cm;
Esto es una caja con 3 cm de padding




Border: Sirve para declarar el borde y sus caracteristicas:

border: 3px solid rgb (0, 0, 0);
Esto es un caja con borde personalizado





Hover: Sirve para cambiar el estilo al pasar el raton por encima:

color: orange;
Esto es un class con :hover





Position absolute: Sirve para cambiar la posicion a cualquier lado de la pantalla:

Position: absolute;
Esto es un div con posicion absoluta
